About My Site

Now I can use JavaScript to implement a simple pagination, an image slider and multiple image albums.

Zina Lu|May 24, 2021


What I have learned

Use JavaScript to generate dynamic html content based on user behavior

Before this project I was building my own website using Jekyll, a static HTML generator with SASS empowered. But I knew nothing about JavaScript because I didn’t have a big motivation to learn it in a way with immense practices.

Wrote five JS files for the website.

Now I can use JavaScript to implement a simple pagination, an image slider and multiple image albums. DOM tree can be flexibly edited by JavaScript and that’s why JavaScript is far more powerful than pure CSS or HTML.


Validation, formatting, and organization

Keeping files and codes in organized is always a good practice, though this is a relatively small website. I find not only does it make later work much more efficient, but also helps me to better understand the progress of website on a more summarized level.


What could be improved

Directories management

My directories are a bit messy and somewhat inefficient. I should place JS files, images, CSS files and other resources into a single directory named “Assets”. It will be much better than letting those four or more directories floating in the root directories.

And I should not place HTML files in main navigation bar in a sub-directory. It will be less error prone if I put those files in the root directory as index file when writing the HTML codes for navigation bar. That is the time I miss Jekyll’s Liquid functionality.

Better artistic polishing

I am not so satisfied with the visual result of my website, though I am really happy with what I have achieved. I love the feedback form and albums. And I think designs of home page and blog page can be improved.
